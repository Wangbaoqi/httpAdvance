# HTTP 导航

关于HTTP，不必多说，对于每一为开发来讲，也是至关重要的。

在此，自己也重新系统的学习HTTP，下面是针对**HTTP**所归纳的脑图，也是一份学习路线。

![](.gitbook/assets/HTTP\_Mind.png)

再次也整理了有关HTTP的规范，能够帮助更深层次的掌握HTTP

* 统一资源定位符 通用语法  [URI RFC - Uniform Resource Identifier Syntax](https://tools.ietf.org/html/rfc3986)
* 超文本传输协议 HTTP/1.0 RFC 1945  [Hypertext Transfer Protocol -- HTTP/1.0](https://tools.ietf.org/html/rfc1945)
* 超文本传输协议 HTTP/1.1 RFC 2616 [Hypertext Transfer Protocol -- HTTP/1.1](https://tools.ietf.org/html/rfc2616)
* 超文本传输协议 HTTP/2.0 RFC 7540  [Hypertext Transfer Protocol -- HTTP/2.0](https://tools.ietf.org/html/rfc7540)

2014年，HTTP/1.1标准 RFC2616 规范内容太过于庞大，将其拆分成了6份小文档，编号为7230 - 7235

1. &#x20;RFC 7230  [Message Syntax and Routing](https://tools.ietf.org/html/rfc7230)
2. &#x20;RFC 7231  [Semantics and Content ](https://tools.ietf.org/html/rfc7231)
3. &#x20;RFC 7232  [Conditional Requests ](https://tools.ietf.org/html/rfc7232)
4. &#x20;RFC 7233  [Range Requests ](https://tools.ietf.org/html/rfc7233)
5. &#x20;RFC 7234  [Caching](https://tools.ietf.org/html/rfc7234)
6. &#x20;RFC 7235  [Authentication](https://tools.ietf.org/html/rfc7235)&#x20;
